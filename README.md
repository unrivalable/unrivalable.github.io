# Unrivalable - Game Website

A cyberpunk arcade-inspired website for the video game **Unrivalable**, part of the **Unemployables** franchise.

## Features

### Opening Screen
- Franchise name "UNEMPLOYABLES" displayed above main title
- Main title "UNRIVALABLE" with glowing neon effects
- "START GAME" option with animated blinking cursor brackets
- CRT monitor aesthetic with scanlines and screen effects

### Home Screen
- Character Selection menu option
- Game Mode menu option
- Start Game button
- Keyboard navigation support (Arrow keys + Enter)
- System status bar at bottom

## Visual Design

- **Cyberpunk arcade aesthetic** with CRT monitor simulation
- **Neon color palette**: Cyan, pink, purple, and green glows
- **Retro-futuristic typography**: Orbitron + Share Tech Mono fonts
- **Dynamic effects**: Scanlines, screen flicker, glitch transitions, chromatic aberration
- **Custom cursor**: Glowing green circular cursor
- **Animated elements**: Pulsing glows, blinking brackets, hover effects

## How to View

Simply open `index.html` in a web browser. For best experience:
- Use a modern browser (Chrome, Firefox, Safari, Edge)
- Full screen recommended for immersive CRT effect
- Desktop viewing recommended (responsive mobile support included)

## Interactions

- **Click "START GAME"** on opening screen to proceed to home screen
- **Hover** over menu items for glow effects
- **Click** menu items to select (logged to console)
- **Keyboard navigation**:
  - Arrow Up/Down to navigate menu
  - Enter to select
  - Enter/Space on opening screen to start

## Tech Stack

- Pure HTML/CSS/JavaScript
- Google Fonts (Orbitron, Share Tech Mono)
- CSS animations and effects
- No external libraries required
